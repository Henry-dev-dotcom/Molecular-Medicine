# Scroll-to-Top Update

## Implemented

- Added a floating up-arrow button to the lower-right corner of the website.
- The button appears after the page is scrolled more than 480 pixels.
- Selecting the button returns the learner to the top of the page.
- Smooth scrolling is used unless the device requests reduced motion.
- Added keyboard focus styling, accessible naming and mobile sizing.
- The control is hidden in printed pages.
- Updated the service-worker cache version so deployed users receive the change.
